# VAYU Website

Official website for VAYU AI Voice Assistant.

## Deploy

1. Upload the `website/` folder to any static host:
   - **GitHub Pages** — push to `gh-pages` branch or a `docs/` folder
   - **Netlify** — drag & drop the folder
   - **Vercel** — deploy the folder directly
   - **Any web server** — copy files to your public HTML directory

2. Place `VAYU_Setup.exe` in `downloads/` folder before deploying.

## Structure

```
website/
├── index.html          # Main landing page
├── README.md
└── downloads/          # EXE installer goes here
```
